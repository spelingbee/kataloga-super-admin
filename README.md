# Super Admin Panel

Панель администратора для управления платформой Kataloga.

## Описание

Super Admin Panel - это централизованная панель управления для администрирования всей платформы. Позволяет управлять арендаторами (tenants), подписками, регистрациями, аналитикой и системными настройками.

## Технологии

- **Nuxt 3** - Vue.js фреймворк
- **TypeScript** - Типизация
- **Pinia** - State management
- **SCSS** - Стили
- **Vite** - Build tool

## Установка

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm dev

# Сборка для production
pnpm build

# Предпросмотр production сборки
pnpm preview
```

## Основные функции

### 1. Управление арендаторами (Tenants)
- Просмотр всех арендаторов
- Активация/деактивация
- Редактирование настроек
- Имперсонация (вход от имени арендатора)

### 2. Регистрации
- Просмотр заявок на регистрацию
- Одобрение/отклонение заявок
- Запрос дополнительной информации
- Просмотр документов

### 3. Подписки
- Управление планами подписок
- Изменение планов
- Продление пробного периода
- Применение скидок
- История платежей

### 4. Аналитика
- Метрики роста
- Анализ выручки (MRR/ARR)
- Географическое распределение
- Когортный анализ
- Воронки конверсии
- Экспорт отчетов (CSV, PDF, Excel)

### 5. Безопасность
- Мониторинг событий безопасности
- Блокировка IP-адресов
- Отслеживание подозрительной активности
- Журнал аудита

### 6. Email управление
- Управление шаблонами писем
- Настройка провайдеров (SMTP, SendGrid, AWS SES)
- История отправленных писем

### 7. Системные настройки
- Настройка платежных шлюзов
- Настройка хранилища (S3, Local)
- Интеграции (Telegram, Webhooks)
- Общие настройки системы

### 8. Поддержка
- Система тикетов
- Объявления для пользователей
- FAQ и документация
- Интерактивные туры

## Переменные окружения

Создайте файл `.env` на основе `.env.example`:

```env
# API
NUXT_PUBLIC_API_BASE_URL=http://localhost:3000/api

# App
NUXT_PUBLIC_APP_NAME=Super Admin Panel
NUXT_PUBLIC_APP_URL=http://localhost:3001
```
